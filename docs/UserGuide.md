# FameHub Teacher & Student User Guide

This guide walks through workflows for instructors (Teachers) and learners (Students).

## Teacher Workflow

### 1. Starting a Virtual Classroom
1. Log in using `teacher@famehub.edu` or your custom teacher credentials.
2. Select **Live Classrooms** on the dashboard side panel.
3. Click **Start Virtual Room**, enter a room name (e.g. "Calculus Lecture 1"), and choose if you want to record the session.
4. Click **Join Class** to launch the classroom interface.

### 2. Monitoring Real-time Participant Logs
The teacher dashboard dynamically displays metrics reflecting the active session:
- Total participants.
- Live active connections.
- Joint join/leave logs updating via WebSockets.

### 3. Assigning Work
- Create and grade assignments.
- Create quizzes with MCQ, true/false, or select-all type questions.

---

## Student Workflow

### 1. Attending Live Lectures
1. Log in using `student@famehub.edu` or custom student credentials.
2. The homepage displays active classes created by your enrolled course teachers.
3. Click **Join Room** to launch the simulator or external classroom window.
4. To stay registered as "Present", stay connected until the threshold timer completes (default: 60s).

### 2. View Grades and Attendance
- Check the Analytics page to view your average grades, total attendance rate, and recent quiz scores.
- Receive instant Toast alerts whenever the teacher posts a new assignment or publishes quiz results.
